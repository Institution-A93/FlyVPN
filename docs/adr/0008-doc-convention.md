# ADR-0008: Конвенция документации (readme.md + claude.md)

- Статус: accepted
- Дата: 2026-06-08

## Контекст
Проект ведём с документацией в каждом каталоге. Нужно разделить человекочитаемое и
машинно-ориентированное, без дублирования.

## Решение
В каждом каталоге — два файла:
- `readme.md` (markdown) — вся проза: что это, зачем, как запустить/проверить.
- `claude.md` (xml) — для агента: цель, инварианты, точки входа, зависимости;
  ссылается на readme через тег `<readme href="./readme.md"/>` и НЕ дублирует прозу.

Канон тегов claude.md зафиксирован в корневом `CLAUDE.md`. Имена файлов в каталогах —
строчными (`readme.md`, `claude.md`); корневые `README.md`/`CLAUDE.md` — как принято
для корня репозитория.

## Последствия
При изменении назначения каталога обновляются оба файла. Агент опирается на claude.md,
человек — на readme.md.
