# ADR-0004: Ingress на Selectel

- Статус: accepted
- Дата: 2026-06-08

## Контекст
RU ingress нужен в российской юрисдикции. У Selectel официальный Terraform-провайдер
(`selectel`), под капотом OpenStack — хорошо ложится на воспроизводимые узлы и
ротацию.

## Решение
Ingress-модуль Terraform пишем первым под Selectel. Модуль делаем по роли (ingress),
провайдер — параметр, чтобы позже добавить других RU-хостеров для разнесения рисков.

## Последствия
Аккаунта Selectel пока нет — модуль пишем под реальный API, применяем по факту
появления доступа.
