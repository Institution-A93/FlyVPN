# ADR-0006: ASN-split роутинг с самого начала

- Статус: accepted
- Дата: 2026-06-08

## Контекст
Рассматривали «весь трафик в туннель» ради простоты против умного split. Split —
главное отличие продукта: скорость для RU-трафика и RU-IP для банков (которые часто
блокируют foreign IP).

## Решение
ASN-split включён в MMVP. На ingress nftables + ipset RU-префиксов: `daddr in
@ru_prefixes` → MASQUERADE напрямую, иначе → mark + таблица mesh → туннель к egress.
Список RU-префиксов обновляется кроном из iptoasn.com / antifilter.network.

## Последствия
Чуть сложнее дата-плейн на узле, но это и есть ценность продукта, а не опция.
