# infra

Infrastructure as Code. Всё облако и конфигурация узлов описаны здесь — узлы
воспроизводимы и являются cattle, не pets. Ротация узла = пересоздание через IaC.

- [`terraform/`](./terraform/readme.md) — провижн облачных ресурсов по ролям.
- [`ansible/`](./ansible/readme.md) — конфигурация поднятых узлов (пакеты, сервисы, шаблоны).

Разделение: OpenTofu создаёт машину и сеть, Ansible приводит её в рабочее состояние.
